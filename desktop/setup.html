Otto Tracker Setup

Otto Tracker keeps patient workflow data local to your office network. The Host still needs occasional internet access for licensing check-ins (no patient data is sent).

Step 1 — Choose Main computer or Client
Runs the database and server locally on this machine. Keep this computer on during business hours. After saving, you’ll complete office setup here.
Does not store the office database locally. If you make changes while disconnected, Otto Tracker may keep a small encrypted offline queue temporarily to sync those changes once the connection returns.
Step 2 — Connect to the Main computer
Get this from the Main computer: File → Show Host Address…
Required the first time you connect. Otto Tracker blocks non-licensing internet access by default.
Step 3 — Sign In
After saving, you’ll sign in or create a login using the Staff code from your office admin.
