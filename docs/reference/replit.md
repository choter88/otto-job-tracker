# Replit-era notes (archived)

This repository originally came from a Replit-hosted deployment. The original `replit.md` mostly documented product features, a Supabase→Postgres migration, and Replit-specific operational details.

It was removed from the repo root during the desktop/offline conversion effort to:
- avoid conflating Replit hosting concerns with the new offline/SOT architecture
- reduce the chance of committing Replit agent state or production export artifacts alongside app code

If you still need historical notes for context (feature inventory, migration artifacts, etc.), recover them from your Git history and copy only the non-PHI parts into `docs/`.
